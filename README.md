# InvRepuestosSC
Proyecto de Inventariado para una Empresa de Repuestos SC

## Cambio de enfoque Fase 1

La Fase 1 ya no considera web scraping.

El requisito principal pasa a ser un proceso ETL para:
- Extraer informacion existente desde archivos Excel de la empresa.
- Transformar y validar datos antes de cargar.
- Cargar registros en una base de datos SQL.
